Annex D

OTShield Visual Capability Overview

Supporting document to OTShield's HMGCC Co-Creation Proposal: Smart Personal Assistant for Security Researchers

D.1 Purpose of this annex

This annex is optional reference material: a visual screenshot pack from the OTShield platform that backs the Phase 1 proposal. Per HMGCC Q&A (Q6), all assessable information is contained in the six-page main proposal; this annex is provided only as fast visual confirmation of features already named and assessed in the main proposal. Reading this annex is not required for evaluation.

The pages that follow are taken from OTShield's standard capability overview document and are referenced from the main proposal:

  • Section 1 (Scope): the "OTShield today" table summarises the live capabilities; the screens in this annex show what those capabilities look like as they run today on the platform.
  • Section 2 (System approach & innovation): the platform foundations described in Annex A (Bundles, Library, Summary, Inventory, Threads, Findings, Vulns, plus the SOC-side decoy fabric, attacker analytics and Purdue topology) are all rendered in the screens overleaf.
  • Annex B (Live threat-intel evidence): the decoy-layer and attacker-TTP screens correspond to the internet-exposed decoy fabric operation that produced the 20-country, 157-attacker, 12,968-attack figures.

D.3 How to read this annex alongside the main proposal

The visual document overleaf is provided as supporting evidence, not as a substitute for any part of the main 6-page proposal. It exists to demonstrate that the live capabilities listed in Section 1 ("OTShield today") and the architectural building blocks described in Section 2 / Annex A are not aspirational: they are operational today, with the screens included.

All figures in the visual document are taken from production OTShield deployments. Any customer-identifying information has been redacted or replaced with representative example data.
